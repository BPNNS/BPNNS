# Bio‑Physical Neural Networks: Navier–Stokes + Madhava–Leibniz + Golden Ratio

This repository contains the code and LaTeX source for the paper  
**“Bio-Physical Neural Networks: A Unified Field Formalism Integrating Navier–Stokes Dynamics, Madhava–Leibniz Spectral Constraints, and Golden Ratio Homeostasis”**  
by Mohsen Mostafa (ORCID: 0009‑0004‑4478‑0317).

## Overview
We simulate a 2D periodic fluid where:
- The velocity field obeys the Navier–Stokes equations forced by a **Madhava–Leibniz gauge field**,
- The synaptic density follows an advection–reaction–diffusion equation with a **golden‑ratio** logistic term,
- The vorticity spectrum is softly constrained to odd wavenumbers with a 1/k² scaling, encoding the Madhava–Leibniz series for π.

Three theoretical predictions are confirmed:
1. **Spectral locking**: energy lives only on odd wavenumbers.
2. **Golden‑ratio homeostasis**: mean synaptic density converges to γ/ϕ ≈ 0.618.
3. **Fibonacci branching**: ridges in the density form angles of 36° (π/5).

## Files
- `code/simulation.py` – main pseudo‑spectral solver.
- `code/requirements.txt` – Python dependencies.
- `paper/biophysical_paper.tex` – LaTeX source of the paper.
- `paper/biophysical_ai_results.pdf` – output figure generated by the simulation.
